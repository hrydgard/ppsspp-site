# The UWP version

UWP, or Universal Windows Platform, is an application packaging standard for apps that can run on both Windows and XBox, and potentially other Windows-based platforms.

## How to run on Xbox using the UWP version

More and better information is coming here in the future.

You need to be able to switch your Xbox One or Series X\|&NoBreak;S into development mode.

There used to be a guide on wiki.xboxdev.store, but currently it's replaced with spam.
